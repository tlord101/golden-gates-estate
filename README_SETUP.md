## ğŸ” Troubleshooting

### "Failed to send email"

1. Check `.env` file has correct Gmail credentials
2. If using Gmail with 2FA, use App Password (not regular password)
3. Run `npm run test:email` to verify SMTP setup

### "Port already in use"

Run on different port:
```bash
PORT=3001 npm start
```

### Email not arriving

- Check spam/junk folder
- Verify recipient email in `.env` (`OWNER_EMAIL`)
- Check server logs for error messages
